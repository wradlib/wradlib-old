wradlib.comp.compose_ko¶

wradlib.comp.compose_ko(radargrids, qualitygrids)¶

Composes grids according to quality information using quality information as a knockout criterion.

The value of the composed pixel is taken from the radargrid whose quality grid has the highest value.

Parameters:
  • radargrids (list of arrays) – radar data to be composited. Each item in the list corresponds to the data of one radar location. All items must have the same shape.
  • qualitygrids (list of arrays) – quality data to decide upon which radar site will contribute its pixel to the composite. Then length of this list must be the same as that of radargrids. All items must have the same shape and be aligned with the items in radargrids.
Returns:

composite (array)
