wradlib.georef.satellite.sat2pol¶

wradlib.georef.satellite.sat2pol(*args, **kwargs)¶

Returns spherical coordinates of PR bins as seen from the GR location.

Deprecated since version 0.11.3: This will be removed in 1.0.0. Use xyz_to_spherical instead.

With PR, we refer to precipitation radars based on space-born platforms such as TRMM or GPM. With GR, we refer to terrestrial weather radars (ground radars).

For this function to work, the pr_xyz coordinates of the PR bins need to be in the azimuthal equidistant projection of the ground radar! This ensures that the ground radar is fixed at xy-coordinate (0, 0), and every PR bin has its relative xy-coordinates with respect to the ground radar site.

New in version 0.10.0.

Parameters:
  • pr_xyz (numpy.ndarray) – Array of shape (nscans, nbeams, nbins, 3). Contains corrected PR xy coordinates in GR azimuthal equidistant projection and altitude
  • gr_site_alt (float) – Altitude of the GR site (in meters)
  • re (float) – Effective Earth radius at GR site (in meters)
Returns:

  • r (numpy.ndarray) – Array of shape (nscans, nbeams, nbins). Contains the slant distance of PR bins from GR site.
  • theta (numpy.ndarray) – Array of shape (nscans, nbeams, nbins). Contains the elevation angle of PR bins as seen from GR site.
  • phi (numpy.ndarray) – Array of shape (nscans, nbeams, nbins). Contains the azimuth angles of PR bins as seen from GR site.
