wradlib.trafo.rvp2dBZ¶

wradlib.trafo.rvp2dBZ(x)¶

Calculates dBZ-values from DWD RVP6 values as given in DX-product files.

Parameters:x (int) – a number or an array

Examples

>>> from wradlib.trafo import rvp2dBZ
>>> print(rvp2dBZ(65.))
0.0
