wradlib.georef.vector.get_shape_points¶

wradlib.georef.vector.get_shape_points(geom)¶

Extract coordinate points from given ogr geometry as generator object

If geometries are nested, function recurses.

New in version 0.6.0.

Parameters:geom (ogr.Geometry) –
Returns:result (generator object) – expands to Nx2 dimensional nested point arrays
