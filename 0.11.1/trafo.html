Data TransformationÂ¶

Module <trafo> transforms data e.g. from RVP-units to dBZ-values to Z-values and vice versa.

rvp2dBZ Calculates dBZ-values from DWD RVP6 values as given in DX-product files.
decibel Calculates the decibel representation of the input values
idecibel Calculates the inverse of input decibel values
r2depth Computes rainfall depth (mm) from rainfall intensity (mm/h)
kdp2r Estimating rainfall intensity directly from specific differential phase.
si2kmh Conversion from SI wind speed units to km/hr.
si2mph Conversion from SI wind speed units to miles/hr
si2kts Conversion from SI wind speed units to knots
kmh2si Conversion from km/hr to SI wind speed units
mph2si Conversion from miles/hr to SI wind speed units
kts2si Conversion from knots to SI wind speed units
ku2s Class to hold coefficients for Radar Reflectivity Conversion
s2ku Class to hold coefficients for Radar Reflectivity Conversion
